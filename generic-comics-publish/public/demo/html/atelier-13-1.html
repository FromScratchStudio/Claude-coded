Expérience HTML embarquée

Atelier 13

Ce chapitre démontre comment la plateforme peut embarquer un épisode HTML distant dans une iframe isolée, sans backend. Il peut servir pour des making-of, bonus interactifs, planches motion ou récits web natifs.

Storyboard liquide

Les séquences peuvent glisser, se superposer et s'animer en toute autonomie dans leur ressource distante.

Modules bonus

Galeries, dossiers d'auteur, paroles de l'équipe et prototypes éditoriaux trouvent leur place dans le même lecteur.

Distribution web

Chaque chapitre HTML est simplement référencé par URL dans le JSON distant, puis chargé à la volée côté client.
